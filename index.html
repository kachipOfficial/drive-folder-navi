📁 Drive フォルダナビ

顧客名・プロジェクト名・フォルダ種別でGoogleドライブを即検索&アップロード

⚙️ 初期設定（初回のみ）

  1. Google Cloud Console にアクセスし、プロジェクトを作成（または既存を選択）
  2. 「APIとサービス」→「ライブラリ」→ Google Drive API を有効化
  3. 「APIとサービス」→「認証情報」→「認証情報を作成」→「OAuthクライアントID」
  4. アプリの種類：ウェブアプリケーション
  5. 「承認済みのJavaScriptオリジン」に http://localhost:8080 を追加して保存
  6. 表示されたクライアントIDをコピーして以下に入力
⚠️ このファイルは python3 -m http.server 8080 で起動したローカルサーバー上で http://localhost:8080/drive-folder-navi.html として開いてください。
file:// で直接開くとGoogle認証が動作しません。
⚠️ Googleにログインしてください

※ どれか1つだけでも検索可。Enterキーでも検索します。

設定をリセット（クライアントIDを変更する場合）
